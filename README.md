# RippleSearchable

Mongoid / Active Record style query criteria and scoping for Ripple.

## Installation

Add this line to your application's Gemfile:

    gem 'ripple_searchable'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ripple_searchable
